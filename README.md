# Data Structures and Algorithms 🔢📊
Welcome to the Data Structures and Algorithms repository! 🎉 This repository contains implementations of various data structures and algorithms in both JavaScript and Java.

The Java implementations of the algorithms were transcribed from the JavaScript versions using OpenAI's GPT-3 language model. 🤖 The readme file you are currently reading was also generated using GPT-3.

The repository is maintained by Raul Peñate, who loves learning about and implementing data structures and algorithms. 🧑‍💻 The website for this repository, built using MKDocs, is primarily based on the master class of Data Structures and Algorithms by Colt Steele. 📚

The website showcases the various algorithms and data structures implemented in this repository, with more to be added in the near future. 🌐 You can access the data by clicking on the `docs` folder above or by visiting our website.

We hope you find this repository useful in your own learning journey! 🚀

In order to set up the project locally, you will need to create a virtual environment and install the dependencies. Here is the process to do so on a Linux machine:

Create a virtual environment:
```python
python3 -m venv venv
```
Activate the virtual environment:
```python
source venv/bin
```

``` python
pip install mkdocs-material
```

The theme will reside in the folder mkdocs-material/material. After cloning from git, you must install all required dependencies with:

```python
pip install -e mkdocs-material
```
